# CentOS 配置Docker国内镜像

编辑配置文件

vi /etc/docker/daemon.json

写入镜像地址

{
  "registry-mirrors": ["https://hub.daocloud.io"]
}

重启服务

systemctl daemon-reload

systemctl restart docker.service

验证

docker info
